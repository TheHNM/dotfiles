      3 This repository contains my config files for a i3-gaps setup.

      5 ## Keybindings for i3 window manager
      6 * `Mod+Enter` Open the terminal
      7 * `Mod+o` Split in horizontal orientation
      8 * `Mod+v` Split in vertical orientation
      9 * `Mod+h` Focus left window
     10 * `Mod+j` Focus bottom window
     11 * `Mod+k` Focus upper window
     12 * `Mod+l` Focus right window
     13 * `Mod+f` Toggle fullscreen for windows
     14 * `Mod+r` Resize windows with vim keybindings
     15 * `Mod+Shift+r` Restart i3 session
     16 * `Mod+Shift+q` Quit __i3__ session
     17 * `Mod+x` Delete clipboard
     18 
     19 ## Configuration and keybindings for installed programs
     20 
     21 My dotfiles contains keybindings for following programs:
     22 * __dmenu__ Open with `Mod+d`

     25 * __firefox__ Open with `Mod+Shift+f`
     26 * __virtualbox__ Open with `Mod+Shift+v`
